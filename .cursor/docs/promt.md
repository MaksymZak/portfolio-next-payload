# Алгоритм выполнения плана

> Полные правила: [`.ai-factory/RULES.md`](../../.ai-factory/RULES.md) — загружаются автоматически в `/aif-implement`.

## Режимы

| Запрос | Поведение |
| --- | --- |
| С номером шага (`/aif-implement 30`, «сделай Step 30») | Только этот шаг |
| Без номера (`/aif-implement`) | Весь план подряд, все невыполненные шаги |

## Краткий чеклист

1. Прочитай план целиком.
2. Выполни шаг(и) по номеру — не перескакивай и не делай побочных улучшений вне scope.
3. Отметь `[x]` только выполненный шаг в plan-файле.
4. При нехватке данных — остановись и спроси (интервью), ничего не выдумывай.
