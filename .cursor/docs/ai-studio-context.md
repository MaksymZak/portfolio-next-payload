# Огляд Проєкту: Портфоліо Frontend Розробника

Це професійне портфоліо для Максима Закалюжного (Maksym Zakaliuzhnyi), Frontend-focused Full Stack Developer. Застосунок створено з фокусом на продуктивність, мінімалістичний строгий дизайн та зручну архітектуру, що може легко розширюватися.

## Особливості та Основний Функціонал

1. **Багатомовність (i18n):**
   * Реалізована внутрішнім керуванням станом (`useState` для перемикання локалей: `en` / `ua`). 
   * Словник текстів винесено до `app/translations.ts`, що робить управління контентом зручним і не засмічує сторінки компонентами.

2. **Дизайн-система (Minimal / Technical):**
   * Максимальний контраст, монохромна палітра з єдиним акцентним кольором (Indigo).
   * Використання сітки (Grid), відсутність тіней та круглих кутів. Більш розгорнуто філософію описано у файлі `DESIGN.md`.

3. **Responsive Макет:**
   * **Desktop:** Розділено на 2 колонки. Ліва частина (інформація / навігація / контакти) є "липкою" (`sticky`), що дозволяє завжди мати під рукою контекст під час прокручування великого масиву проєктів чи навичок.
   * **Mobile:** Плавно інтегрується в одну колонку, із винесеним липким `Bottom Header` та вспливаючим Drawer-меню для навігації.

4. **Версія резюме для друку (Print-ready CV):**
   * Застосунок має окремий маршрут `/cv`, який оптимізовано не тільки під веб-перегляд, а й розроблено за допомогою Tailwind-класів `print:*` спеціально для друку.
   * Нажавши "SAVE PDF", користувач отримує ідеально адаптовану версію документа без елементів інтерфейсу сайту (top bar приховується, темні блоки знебарвлюються для нормальної якості на принтері).

5. **Дані та Масштабованість:**
   * Проєкти та навички розділені на конкретні логічні блоки (Experience, Soft Skills, Stack). 
   * Додано "архівний" розділ (`/portfolio-cms`) як приклад розширення платформи, а загальна структура коду орієнтується на чистоту компонентів і строгу типізацію (`TypeScript`).

## Стек технологій

* **Фреймворк:** Next.js (App Router)
* **Мова:** TypeScript
* **Стилізація:** Tailwind CSS 
* **Іконки:** Lucide React

## Локальна розробка

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Запустіть dev-сервер:
   ```bash
   npm run dev
   ```
3. Відкрийте застосунок за адресою `http://localhost:3000`.

## Розгортання (Deployment)

Застосунок готовий для Production-білду. `next build` компілює статичний сайт та набір Server Components:
```bash
npm run build
npm start
```
